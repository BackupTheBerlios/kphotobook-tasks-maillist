Status: Open
Authority  : chtom
Assigned to: 

Description: if an image is tagged with a boolean tag this image implicitely is tagged also with all parents of this boolean tag.

example: 

Switzerland        [  ]    [x]
      Aargau         [  ]    [  ]
             Aarau    [x]    [  ]   

An image tagged with Aarau must also be found if we are searching for images tagged with Aargau or Switzerland.
This feature can be enabled/disabled.
